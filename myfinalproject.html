Welcome

This project is a comprehensive exploration of Universal Design and how it shapes accessible, usable, and meaningful digital experiences. Over the semester we learned that accessibility is not an add-on but a foundational part of creating technology that works for everyone. This website brings together those ideas in a clear, structured way while demonstrating the same principles it teaches.

What This Site Covers

Universal Design goes beyond meeting minimum accessibility standards. It focuses on designing for human diversity from the beginning so that digital spaces are inclusive to people of all abilities, devices, and environments. In this project you will learn:

  • What Universal Design is and how it developed over time
  • Why Universal Design matters for users, creators, and the web as a whole
  • How to apply these principles using techniques like semantic HTML, responsive design, accessible forms, ARIA roles, and WCAG guidelines

How to Navigate This Project

The navigation at the top of the page will guide you through the main sections. Each page focuses on a different aspect of Universal Design and presents researched, course-based content supported by examples, best practices, and accessibility techniques.

  • What: Defines Universal Design and introduces the principles behind it
  • Why: Explains the need, impact, and history behind inclusive design
  • How: Shows practical steps and tools for implementing Universal Design in real projects

Project Purpose

My goal for this final project was to create a site that does more than present information. It demonstrates accessible structure, semantic HTML, logical navigation, and responsive design all key ideas emphasized in this course. Every page aims to reflect Universal Design in practice as well as in theory.
